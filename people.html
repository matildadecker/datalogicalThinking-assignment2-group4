The team

This page briefly summarize how we organized our collaboration, who did what, and witch tools we used while building MovieMood

Team communication

Our initial coordination took place by Canvas messages, where we scheduled a Zoom meeting with all four group members. Matilda, Johanna and Cajsa attended the meeting and brainstormed ideas for the project setup and structure. We then created a Discord channel for all ongoing communication, using both text and voice. Active members regularly shared progress updates and gave each other feedback.

Responsibilities

We divided the work so that Matilda was responsible for the dataset, while Johanna and Cajsa were responsible for the HTML pages. We coordinated closely, asked for help when needed, and reviewed each other's work to keep the style and structure consistent across the site.

Team Members

  • Matilda — Dataset curation and structure.
  • Johanna — HTML layout, page structure, and content.
  • Cajsa — HTML pages, content and overall styling.
  • Afroditi — Not present.

Tools Used

  • GitHub — version control and collaboration
  • Zoom — planning meeting
  • Discord — daily communication (chat and calls)
  • Google drive - taking notes and planning documentation
  • VS Code — HTML/CSS editing
