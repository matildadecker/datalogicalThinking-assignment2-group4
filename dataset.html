The dataset

We started by downloading the TMDB 5000 Movie Dataset from Kaggle, which contains information about approximately 5000 movies. The dataset was downloaded as a CSV file. Since we didnâ€™t need all 5000 movies, we decided to reduce the dataset to only 100 movies using the bash terminal. We used the following command:

head -n 101 tmdb_5000_credits.csv > tmdb_100.csv

This command copies the first 101 lines of the original file, one header line and 100 data rows, and saves them in a new file called tmdb_100.csv. Next, we wanted to remove the crew column since it wasnâ€™t relevant for our project. To do that, we used the command:

csvcut -c movie_id,title,cast movies_100.csv > movies_100_clean.csv

After cleaning the data, we wanted to have it available in different formats. We uploaded the CSV-file to Google Sheets, then installed the Export Sheet Data add-on. With this tool, we exported the same dataset as both XML and JSON files. This made it easier to use the data in different parts of our web project and to test how various data formats could be implemented.
